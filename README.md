# <u>**Craigslist Dublin Test Framework**</u>

# <u>**Tools and Technologies:**</u>

 **1. Language: Java**

 **2. Testing framework: Testng**

 **3. BDD framework: Cucumber jvm**

 **4. Automation tool: Selenium webdriver**

 **5. Build tool: maven**

 **6. Reporting: cucumber reporting**

# <u>**Running Tests:**</u>

**$ mvn clean install**
