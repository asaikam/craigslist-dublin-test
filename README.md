
#Craigslist Dublin Test Framework

##Tools and Technologies:

 **1. Language: Java**

 **2. Testing framework: Testng**

 **3. BDD framework: Cucumber jvm**

 **4. Automation tool: Selenium webdriver**

 **5. Build tool: maven**

 **6. Reporting: cucumber reporting**

##Running Tests:
**$ mvn clean install**
